# 📈 Stock Fundamentals Cheat Sheet

An interactive, high-performance, dark-themed **Stock Fundamentals Cheat Sheet & Screener Companion Guide** built to help investors quickly evaluate public companies on financial platforms like **Screener.in**, **ET Markets**, and **Moneycontrol**.

---

## 🚀 Quick Start

1. Open `index.html` in **Google Chrome**, **Microsoft Edge**, or any modern web browser.
2. Use the **Search Bar** (or press `Ctrl + K`) to search any ratio (e.g., `ROE`, `Debt`, `PEG`, `Promoter Pledge`).
3. Use the **60-Second Checklist** while viewing a stock on Screener.in to instantly score the company's quality.

## ✨ Key Features & Sections

### 1. 🏠 Dashboard Home & Legend
- **Color Legend System**:
  - 🟢 **Excellent**: Industry-leading, ultra-safe performance.
  - 🟢 **Good**: Healthy, robust fundamental profile.
  - 🟡 **Acceptable / Average**: Moderate performance, investigate further.
  - 🟠 **Caution**: Watch out for potential weakness.
  - 🔴 **Red Flag**: Immediate dealbreaker / exit rule.

### 2. ⚡ 60-Second Interactive Checklist
Interactive 8-point pass/fail checklist:
- □ Revenue growing? (>10% YoY / multi-year CAGR)
- □ Profit growing? (PAT & EPS growing faster than sales)
- □ ROE >15%? (High return on equity)
- □ ROCE >15%? (High return on capital employed)
- □ Debt manageable? (Debt/Equity < 0.5 or Interest Coverage > 5x)
- □ Positive Cash Flow? (Operating Cash Flow > Net Profit)
- □ Promoter pledge = 0%? (No mortgaged shares)
- □ Valuation reasonable? (PEG < 1.0 or fair historical P/E)

Provides real-time score calculation and stock quality verdict.

### 3. 📊 Metric Detail Cards
Covers 35+ fundamental metrics across 7 core categories:
1. **Growth**: Revenue Growth, Sales CAGR, Profit Growth, EPS Growth, Quarterly Growth.
2. **Profitability**: ROE, ROCE, Operating Margin (OPM), EBITDA Margin, Net Margin.
3. **Financial Health**: Debt to Equity, Current Ratio, Interest Coverage.
4. **Valuation**: P/E, PEG Ratio, Price to Book (P/B).
5. **Ownership**: Promoter Holding, Promoter Pledge.
6. **Cash Flow**: Operating Cash Flow (OCF), Free Cash Flow (FCF).
7. **Quality & Governance**: Economic Moat & Brand, Corporate Governance & Audit Red Flags.

**Every metric card includes:**
- What is it?
- Ideal target range
- Color-coded interpretation matrix table
- Higher or Lower indicator (⬆ / ⬇)
- Why investors care
- Watch out! warnings
- Common beginner mistakes
- Quick memory tricks
- Cross-reference comparison tags
- Sector variation badges

### 4. 🛠 Quick Stock Analyzer (Screener Helper)
Type numbers directly from Screener.in or ET Markets into the interactive widget to get an immediate automated fundamental verdict and breakdown.

### 5. 🔀 Company Evaluation Flowchart
Visual step-by-step decision tree guiding your analysis from top-line revenue down to qualitative moat evaluation.

### 6. 🔴 Dedicated Biggest Red Flags Hub
Catalog of 8 critical deal-breakers (Pledged shares, negative OCF, auditor resignation, channel stuffing receivables, equity dilution, etc.).

### 7. 🏢 Bonus Sector Exceptions Guide
Matrix table detailing sector-specific metric rules for **Banks**, **Insurance**, **IT & SaaS**, **FMCG**, **Utilities**, **Airlines**, **Real Estate**, and **Capital Goods**.
